# LUCKY 🐾

## Problem
Türkiye'de ruh sağlığı hâlâ büyük bir sosyal damga taşıyor. 
İnsanlar yardım aramaktan çekiniyor. Türkçe, sıcak ve 
yargılamayan bir alan yok.

## Çözüm
LUCKY, nasıl hissettiğini yazdığında sana yargılamadan 
karşılık veren, ruh haline özel şiir ve renk paleti üreten, 
küçük bir aktivite öneren dijital bir dost.
