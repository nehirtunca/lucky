# LUCKY — Teknoloji Yığını (Tech Stack)

## Seçilen Teknolojiler

| Teknoloji | Neden Seçtik? |
|-----------|---------------|
| HTML5 | Web'in iskeletidir, her tarayıcıda çalışır |
| CSS3 | Uygulamanın duygusunu tasarlamak için |
| Vanilla JavaScript | Gemini API ile konuşmak ve ekranı güncellemek için |
| Gemini API | Ücretsiz, Türkçe desteği mükemmel, kurulumu hızlı |
| Vercel / GitHub Pages | Tek tıkla ücretsiz yayınlama |

## Neden Framework Kullanmıyoruz?
Başlangıç seviyesi için en sade yaklaşım. React veya 
Vue gibi büyük yapılar öğrenme sürecini yavaşlatır. 
Biz doğrudan tarayıcının ana dilini konuşacağız.

## Kurulum Adımları

### 1. VS Code veya Cursor
- cursor.com adresinden Cursor'ı indir ve kur
- "Live Server" eklentisini yükle

### 2. Proje Klasörü
- Masaüstünde `LUCKY-APP` klasörü oluştur
- İçine 3 boş dosya ekle:
  - `index.html`
  - `style.css`
  - `script.js`

### 3. Gemini API Anahtarı
- aistudio.google.com adresine git
- Google hesabınla giriş yap
- "Get API Key" → "Create API key" tıkla
- Anahtarı güvenli bir yere kaydet, kimseyle paylaşma!

### 4. İlk Test
- index.html dosyasını Cursor'da aç
- Live Server'ı başlat
- Tarayıcıda boş sayfa görünüyorsa hazırsın!
